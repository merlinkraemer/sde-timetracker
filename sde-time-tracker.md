# SDE Time Tracker

Skydive Events (SUB-events GmbH) time tracking.

## Usage
Send start/stop times to Clawd:
- "start sde" / "stop sde - note about what you did"
- Or send specific times: "sde 14:00-18:30 - deployed auth fix"
